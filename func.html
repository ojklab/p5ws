関数（ 関数定義 と 関数呼び出し ）

関数とは、一連の処理や計算をまとめてひとつのまとまりにしたものです。
関数の記述は「 ① 関数定義 」と「 ② 関数呼び出し 」に大別され、意味は以下の通りです！

① 関数定義 ： 関数が呼び出されたときに実行される処理（関数の中身）を記述すること
② 関数呼び出し ： 定義した関数の中身を呼び出して、実行させること

【 Pythonの場合 】
① 関数定義 ： def 関数（） :
② 関数呼び出し ： 関数名()

【 JavaScriptの場合 】
① 関数定義 ： function 関数名() {
　 　 　 　 　　}
② 関数呼び出し ： 関数名();

def draw() :
　sample()　// 関数呼び出し

def sample() :　// 関数定義
　fill(0, 0, 255)
　square(150, 200, 100)
　fill(255, 255, 0)
　triangle(200, 120, 280, 200, 120, 200)
function draw() {
　sample() ;　//関数呼び出し
}

function sample() {　//関数定義
　fill(0, 0, 255) ;
　square(150, 200, 100) ;
　fill(255, 255, 0) ;
　triangle(200, 120, 280, 200, 120, 200) ;
}

引数ありの関数

関数に引数を指定することで、関数の振る舞いを制御することができます。
引数は、数値や文字列を直接指定するだけでなく、変数や式で指定することもできます。

【 Pythonの場合 】
関数定義 ： def 関数（引数, 引数, ...） :
関数呼び出し ： 関数名(引数, 引数, ...)

【 JavaScriptの場合 】
関数定義 ： function 関数名(引数, 引数, ...) {
　　 　 　 　}
関数呼び出し ： 関数名(引数, 引数, ...) ;

↑ 引数のない関数と比べて、関数の後ろに付いている（）の中に引数が入っただけです！

def draw() :
　sample(150, height / 2) 　//関数呼び出し

def sample(x,y) : 　//関数定義
　 circle(x, y, 50)
function draw() {
　sample(150, height / 2) ;　//関数呼び出し
}

function sample(x,y) {　//関数定義
　circle(x, y, 50) ;
}

戻り値ありの関数

関数を実行して得られる結果のことを「戻り値（返り値）」と呼びます。

【 Pythonの場合 】
def 関数名(引数, 引数, ...) :　← 関数定義
　処理の内容
　return 戻り値
～どこか別の場所～
関数名(引数, 引数, ... )　← 関数呼び出し

【 JavaScriptの場合 】
function 関数名(引数, 引数, ... ) {　← 関数定義
　処理の内容
　return 戻り値 ;
}
～どこか別の場所～
関数名(引数, 引数, ... ) ;　← 関数呼び出し

def draw() {
　print(calcBMI(1.65, 55))　//関数呼び出し
}

def calcBMI(h, w) {　//関数定義
　return w / (h * h)
}
function draw() {
　print(calcBMI(1.65, 55)) ; 　//関数呼び出し
}

function calcBMI(h, w) {　//関数定義
　return w / (h * h) ;
}
