if 文

条件分岐とは、「プログラムにおいて指定した条件によって、処理の流れを変更することができる制御構造」のことです。
つまり、ある条件が真であれば、それに沿った処理を行い、偽であれば別の処理を行うようにプログラムを設定できます。

【Pythonの場合】
if 条件式 :
　条件を満たした場合に実行したいコード

【JavaScriptの場合】
if (条件式) {
　条件を満たした場合に実行したいコード
}
となります。

Javascriptでは、if の後ろに ( ) と { } が続き、( ) の中に 条件式を書き、その条件が満たされたときに実行されるコードを { } の中に書きます。

dia = 100

if dia == 100 :
circle(width / 2, height / 2, dia)
let dia = 100 ;

if (dia == 100) {
　circle(width / 2, height / 2, dia) ;
}

if-else 文

続いて、if-else文は「もしAならばBを行う、そうでなければCを行う」という条件分岐です。

【 Pythonの場合 】
if 条件式 :
　条件を満たした場合に実行したいコード
else :
　当てはまらない場合に実行したいコード

【 JavaScriptの場合 】
if (条件式) {
　条件を満たした場合に実行したいコード
} else {
　当てはまらない場合に実行したいコード
}
となります。

dia = 180

if dia >= 100 :
　fill (0, 0, 255)　//青色
else :
　fill (255, 255, 0)　//黄色

circle (width / 2, height / 2, dia)
let dia = 180 ;

if (dia >= 100) {
　fill(0, 0, 255) ;　//青色
} else {
　fill(255, 255, 0) ;　//黄色
}

circle(width / 2, height / 2, dia);

else-if 文

if文に加えて、条件が falseの場合に別の処理を実行するための方法として「else-if」文があります。
else文とは違い、else-if文では新しい条件を指定でき、その条件がtrueであれば、その条件に対応する処理が実行されます。

たとえば、if文ではある一つの条件しかチェックしませんが、else-if文を使うことで、複数の条件を順番にチェックし、最初にtrueになった条件に対応する処理を実行できます。

簡単に言えば、「もし～だったら、こうする。それがダメなら、これを試す。それも違うなら、これをする」というように、条件ごとに異なる処理を行いたいときに使います。

【 Pythonの場合 】
if 条件1 :
　条件を満たした場合に実行したいコード
elif 条件2 :
　当てはまらない場合に実行したいコード
else :
　上記に当てはまらない場合に実行したいコード

【 JavaScriptの場合 】
if (条件1) {
　条件1を満たした場合に実行したいコード
} else if (条件2) {
　条件2を満たした場合に実行したいコード
} else {
　上記に当てはまらない場合に実行したいコード
}
となります。
