今回のワークショップでは、ここで紹介する「テンプレート」として用意された関数を使います。パラメータが多くてちょっとややこしいですが、基本がわかればどのテンプレートもおおよそ同じです。

テンプレートの使い方

テンプレートは６種類！

リバウンド（ws_rebound）
複数の図形がランダムに動き回って、キャンバスの端で跳ね返ります。
パルス（ws_pulse）
複数の図形がランダムに出現しては消えていきます。
ウィール（ws_whirl）
複数の図形が回転しながら、その輪を収縮あるいは膨張させます。
スパイラル（ws_spiral）
単体の図形が円を描きながら中心あるいは外側に向かって動きます。ウィールの単体図形版です。
グリッド（ws_grid）
キャンバスに敷き詰められた図形の色が変化します。
ライン（ws_line）
複数の直線を任意の角度で移動させることができます。

使い方の基本

一番シンプルな使い方は、次のように、テンプレートの関数の引数に { } だけを書いて呼び出すことです。

ws_rebound({});

この { } の中に「パラメータ名: 値」という形式でパラメータを指定することで、図形の数や大きさ、丸み、動き、色などを変更することができます。

ws_rebound({ size: 70 });

指定できるパラメータや値はテンプレートによって変わります。このウェブページの残りの部分は個別のテンプレートのパラメータの説明になっています。

パラメータは、カンマで区切って複数指定することができます。

ws_rebound({ num: 5, size: 70, R: 0.7 });

長くなるときは途中で改行しても構いません（単語や数値の途中でなければどこで改行してもOK）。

ws_rebound({
　num: 5,
　size: 70,
　R: 0.7
});

ちなみに、指定されていないパラメータには「既定値」が裏で指定されているので、パラメータを何も指定しなくてもとりあえず動きます。

パラメータの値

パラメータの値には「数値」と「文字列」が使えます。数値は「3」といったように半角の数字で、文字列は「"size"」といったように引用符で囲います。

ws_pulse({ size: 50, fade: "opacity" });

パラメータによっては、２つの値を指定できるものがあります。その場合は [ ] で囲って、カンマを挟んで値を並べます。

ws_whirl({ size: [50, 100], speed: [5, 10] });

ウィール（ws_whirl）の場合、sizeに50と100という2つの数値を与えると、50から100のあいだの乱数（ランダムな数）が割り当てられます。また、speedに与えられた2つの値（5と10）は、1つ目が回転速度、2つ目が収縮速度…というように意味が異なります。2つの値がどういう意味になるのかはテンプレート毎に表にまとめています。

色の指定のみ特殊で、次のように [ ] の中に「color("色名")」という形式でいくつでも指定できます。1色しか指定しなければ必ずその色に、2色以上指定したらランダムに選ばれます。

ws_pulse({ cols: [color("khaki"), color("pink"), color("skyblue")] });

使える色の名前は こちら などを参照してください。広告ありでよければ「ウェブ 色」で検索すればもっと見やすいサイトもあります。

もどる

その他のルール

  1. 異なる種類のテンプレートは同時に使えます。後から呼び出したほう（コードの下に書いたほう）が上に描かれます。
  2. 同じテンプレートのパラメータを変更して呼び出しても、一番最初に呼び出したパラメータ設定のままで動き続けます。パラメータを変化させたい場合は、「ws_reset("テンプレート名");」という関数を呼び出す必要があります。このとき、テンプレート名には「ws_」は含まないので注意してください。
  3. たくさんのテンプレートを呼び出すほど、また、パラメータ num で指定できる図形数を増やすほど、処理が遅くなります。PCの能力に合わせて調整してください。
  4. パラメータの名前や値を1文字でも間違えると正しく動きません。間違えてもエラーにならず、スルーされて既定値のまま動くことが多いので、おかしいなと思ったら間違い探しをしてください。
  5. テンプレートに用意された図形の形状は、残念ながら、円〜角丸〜正方形と直線しかありません。しかし、p5.jsを勉強してきた人は自分でp5.jsの関数を呼び出して、自由に図形を追加することができます。

リバウンド（ws_rebound）

複数の図形がランダムに動き回って、キャンバスの端で跳ね返ります。

個々の図形の大きさ・丸み・移動速度・色・不透明度は、乱数形式で指定することで、個別に変更することができます。最もシンプルなテンプレートなので、このテンプレートでパラメータ指定の練習をするとよいでしょう。

パラメータ名 値の形式 意味 既定値 乱数
num 整数値 図形の数 5 不可
size 数値 各図形の大きさ [20, 80] 可
R 0.0〜1.0 各図形の丸み（1が円） 1.0 可
cols 参照 各図形の色 - -
opacity 0.0〜1.0 各図形の不透明度（0が透明） 1.0 可
speed 数値 各図形の移動速度 [2, 5] 可

ws_rebound({ num: 7, size: [30,70], R: [0.5,1] });

ws_rebound({ speed: [2, 15], cols: [color("pink")], opacity:[0.5, 1] });

パルス（ws_pulse）

複数の図形がランダムに出現しては消えていきます。

個々の図形の大きさ・丸み・変化速度・色・不透明度は、乱数形式で指定することで、個別に変更することができます。パラメータ fade で、図形の消え方を「小さくなって消える」と「色が薄くなって消える」から選べます。

パラメータ名 値の形式 意味 既定値 乱数
num 整数値 図形の数 5 不可
size 数値 各図形の大きさ [20, 80] 可
R 0.0〜1.0 各図形の丸み（1が円） 1.0 可
cols 参照 各図形の色 - -
opacity 0.0〜1.0 各図形の不透明度（0が透明） 1.0 可
speed 数値 各図形の消失速度（"size"） [1, 2] 可
各図形の消失速度（"opacity"） [5, 20]
fade 文字列 "size" … 小さくなって消える
"opacity" … 薄くなって消える		 "size" 不可

ws_pulse({ num: 9, size: [40,70], R: [0.5,1], speed: [5,15] });

ws_pulse({ cols: [color("orange")], opacity: [0.5, 1], fade: "opacity" });

ウィール（ws_whirl）

複数の図形が回転しながら、その輪（円環）を収縮あるいは膨張させます。

個々の図形の大きさ・丸み・色・不透明度は、乱数形式で指定することで、個別に変更することができます。各種速度（speed）や円環の大きさ（diameter）は乱数になりません。
その他、ウィールの場合、以下のようなパラメータがあります。

  • 速度のパラメータ speed に2つの値を指定すると、個々の図形の速度をランダムに変化させるのではなく、円環の [回転速度, 収縮／膨張速度] という意味になります。値が1つだけの場合は回転速度と収縮／膨張速度は同じ値になります。
  • パラメータ direction は、円環の収縮／膨張の方向もしくは 左右の回転方向を表す4種類の文字列を指定します。2つの値を指定すると、[円環の収縮／膨張の方向, 左右の回転方向] という意味になります。
  • パラメータ fluctuate は、収縮と膨張を切り替えて繰り返すか、収縮か膨張のみを繰り返すかを指定します。
パラメータ名 値の形式 意味 既定値 乱数
num 整数値 図形の数 6 不可
size 数値 各図形の大きさ 50 可
R 0.0〜1.0 各図形の丸み（1が円） 1.0 可
cols 参照 各図形の色 - -
opacity 0.0〜1.0 各図形の不透明度（0が透明） 1.0 可
diameter 数値 円環の最大値 キャンバスサイズ 不可
speed 数値 速度（回転と収縮／膨張は同値） 8 不可
[数値, 数値] 回転速度 と 収縮／膨張速度
direction 文字列 "inf" … 膨張、"def" … 収縮
"right" … 右回り、"left" … 左回り		 ["def", "right"] 不可
[文字列, 文字列] ["inf"/"def", "right"/"left"]
fluctuate 文字列 "on" … 収縮と膨張を切り替える
"off" … 収縮／膨張のみを繰り返す 		"on" 不可

ws_whirl({ num: 7, size: [30, 80], R: [0.5, 1], speed: [5, 18], cols: [color("green")] });

ws_whirl({ opacity: [0.5, 1], direction: ["inf", "left"], fluctuate: "off", diameter: 200});

スパイラル（ws_spiral）

単体の図形が円を描きながら中心あるいは外側に向かって動きます。ウィールの単体図形版です。

図形が単体なので、図形の大きさ・丸み・不透明度の乱数指定はできません。その他、ウィールと同様、以下のようなパラメータがあります。

  • 速度のパラメータ speed に2つの値を指定すると、個々の図形の速度をランダムに変化させるのではなく、図形の [回転速度, 収縮／膨張速度] という意味になります。指定する値が1つだけの場合は回転速度と収縮／膨張速度は同じ値になります。
  • パラメータ direction に2つの値を指定すると、[図形の収縮／膨張の方向, 左右の回転方向] という意味になります。指定する値が1つだけの場合は「収縮／膨張方向」を意味します。
  • パラメータ interval はウィールにはないパラメータで、図形の色を円運動の途中で変化させる間隔（秒数）を指定できます。このパラメータを省略した場合、円運動の途中で色は変化しません。
  • ウィールとは異なり、パラメータ fluctuate はありません。
パラメータ名 値の形式 意味 既定値 乱数
size 数値 図形の大きさ 50 不可
R 0.0〜1.0 図形の丸み（1が円） 1.0 不可
cols 参照 図形の色 - -
opacity 0.0〜1.0 図形の不透明度（0が透明） 1.0 不可
diameter 数値 円環の最大値 キャンバスサイズ 不可
speed 数値 速度（回転と収縮／膨張は同値） 10 不可
[数値, 数値] 回転速度 と 収縮／膨張速度
direction 文字列 "inf" … 膨張、"def" … 収縮
"right" … 右回り、"left" … 左回り		 ["def", "right"] 不可
[文字列, 文字列] ["inf"/"def", "right"/"left"]
interval 数値 図形の色がスパイラルの途中で変化する間隔（秒数） 0（途中変化なし） 不可

ws_spiral({ size: 70, R: 0, speed: [80, 10], cols: [color("skyblue")] });

ws_spiral({ opacity: 0.9, direction: ["def", "right"], diameter: 150 });

グリッド（ws_grid）

キャンバスに敷き詰められた図形の色が変化します。また、パラメータ noise を指定することによって大きさが細かく変化してプルプル振動します。

個々の図形の大きさ・丸み・変化速度・色・不透明度は、乱数形式で指定することで、個別に変更することができます。

個数のパラメータ num に指定した値は「図形の数」ではなく「1辺の図形の数」になります（つまり、num×num個の図形が描かれる）。また、パラメータ interval で各図形の色が変わる間隔（秒数）を指定できます。

パラメータ名 値の形式 意味 既定値 乱数
num 整数値 一辺の図形の数 5 不可
size 数値 各図形の大きさ numに依存 可
R 0.0〜1.0 各図形の丸み（1が円） 1.0 可
cols 参照 各図形の色 - -
opacity 0.0〜1.0 各図形の不透明度（0が透明） 1.0 可
interval 数値 各図形の色が変わる間隔（秒） [0.5, 2] 可
noise 数値 各図形の振動の大きさ 0（振動しない） 可

ws_grid({ size: [1, 40], R: [0.5 ,1], cols: [color("purple")] });

ws_grid({ opacity: 0.7, interval: 0.5, noise: 5 });

ライン（ws_line）

複数の直線を任意の角度で移動させることができます。

個々の直線の長さ・太さ・移動速度・色・不透明度は、乱数形式で指定することで、個別に変更することができます。ラインでは、図形の数（パラメータ num）は目安です（正確な数にならない）。

直線が移動方向（向かう先）は、パラメータ angle で度数（degree）指定します。12時の方向が 0 で、6時の方向が 180 です。すべての直線は同じ方向に移動します（個別にランダム指定はできない）。

パラメータ名 値の形式 意味 既定値 乱数
num 整数値 直線の（おおよその）数 20 不可
size 数値 各直線の長さ [20, 80] 可
weight 数値 各直線の太さ 2 可
cols 参照 各直線の色 - -
opacity 0.0〜1.0 各直線の不透明度（0が透明） 1.0 可
speed 数値 各直線の移動速度 [10, 30] 可
angle 0〜360 直線の進行方向（角度） 90（3時の方向） 不可

ws_line({ num: 50, size: [1, 100], weight: [5, 30], speed: [20, 60] });

ws_line({ cols: [color("green")], opacity: 0.8, angle: 45 });
