今回のワークショップでは、ここで紹介する「テンプレート」として用意された関数を使います。パラメータが多くてちょっとややこしいですが、基本がわかればどのテンプレートもおおよそ同じです。

テンプレートの使い方

テンプレートは６種類！

① ws_rebound({}); // リバウンド
② ws_pulse({}); // パルス
③ ws_whirl({}); // ウィール
④ ws_spiral({}); // スパイラル
⑤ ws_grid({}); // グリッド
⑥ ws_line({}); // ライン

使い方の基本

一番シンプルな使い方は、次のように、テンプレートの関数の引数に { } だけを書いて呼び出すことです。

ws_rebound({});

この { } の中に「パラメータ名: 値」という形式でパラメータを指定することで、図形の数や大きさ、丸み、動き、色などを変更することができます。

ws_rebound({ size: 70 });

指定できるパラメータや値はテンプレートによって変わります。このウェブページの残りの部分は個別のテンプレートのパラメータの説明になっています。

パラメータは、カンマで区切って複数指定することができます。

ws_rebound({ num: 5, size: 70, R: 0.7 });

ちなみに、指定されていないパラメータには「既定値」が裏で指定されているので、パラメータを何も指定しなくてもとりあえず動きます。

パラメータの値

パラメータの値には「数値」と「文字列」が使えます。数値は「3」といったように半角の数字で、文字列は「"size"」といったように引用符で囲います。

ws_pulse({ size: 50, fade: 'opacity' });

パラメータによっては、２つの値を指定できるものがあります。その場合は [ ] で囲って、カンマを挟んで値を並べます。

ws_whirl({ size: [50, 100], speed: [5, 10] });

ウィール（ws_whirl）の場合、sizeに50と100という2つの数値を与えると、50から100のあいだの乱数（ランダムな数）が割り当てられます。また、speedに与えられた2つの値（5と10）は、1つ目が回転速度、2つ目が収縮速度…というように意味が異なります。2つの値がどういう意味になるのかはそれぞれ説明があります。

色の指定のみ特殊で、次のように [ ] の中に「color('色名')」という形式でいくつでも指定できます。1色しか指定しなければ必ずその色に、2色以上指定したらランダムに選ばれます。

ws_pulse({ cols: [color('khaki'), color('pink'), color('skyblue')] });

使える色の名前は こちら などを参照してください。広告ありでよければ「ウェブ 色」で検索すればもっと見やすいサイトもあります。

その他のルール

  1. 異なる種類のテンプレートは同時に使えます。後から呼び出したほう（コードの下に書いたほう）が上に描かれます。
  2. 同じテンプレートのパラメータを変更して呼び出しても、一番最初に呼び出したパラメータ設定のままで動き続けます。パラメータを変化させたい場合は、「ws_reset('テンプレート名');」という関数を呼び出す必要があります。このとき、テンプレート名には「ws_」は含まないので注意してください。
  3. たくさんのテンプレートを呼び出すほど、また、パラメータ num で指定できる図形数を増やすほど、処理が遅くなります。PCの能力に合わせて調整してください。
  4. パラメータの名前や値を1文字でも間違えると正しく動きません。間違えてもエラーにならず、スルーされて既定値のまま動くことが多いので、おかしいなと思ったら間違い探しをしてください。
  5. テンプレートに用意された図形の形状は、残念ながら、円〜角丸〜正方形と直線しかありません。しかし、p5.jsを勉強してきた人は自分でp5.jsの関数を呼び出して、自由に図形を追加することができます。

リバウンド（ws_rebound）

複数の図形がランダムに動き回って、キャンバスの端で跳ね返ります。

個々の図形の大きさ・丸み・移動速度・色・不透明度は、乱数形式で指定することで、個別に変更することができます。最もシンプルなテンプレートなので、このテンプレートでパラメータ指定の練習をするとよいでしょう。

① num: 数値 → 図形の数
② ★size: 数値 → 図形の大きさ（1辺あるいは直径の長さ）
③ ★R: 数値 → 角の丸さ（0～1）0が四角、1が円
④ ★speed: 数値 → 図形の動く速さ
⑤ cols: [color('色の名前')] → 図形の色
⑥ ★opacity: 数値 → 不透明度（0～1）0が透明、1が不透明
★マークのついたパラメータは [最小値, 最大値] で乱数になる

ws_rebound({ num: 7, size: [30,70], R: [0.5,1] });

ws_rebound({ speed: [2, 15], cols: [color('pink')], opacity:[0.5, 1] });

パルス（ws_pulse）

複数の図形がランダムに出現しては消えていきます。

個々の図形の大きさ・丸み・変化速度・色・不透明度は、乱数形式で指定することで、個別に変更することができます。パラメータ fade で、図形の消え方を「小さくなって消える」と「色が薄くなって消える」から選べます。

① num: 数値 → 図形の数
② ★size: 数値 → 図形の大きさ（1辺あるいは直径の長さ）
③ ★R: 数値 → 角の丸さ（0～1）0が四角、1が円
④ ★speed: 数値 → 図形の動く速さ
⑤ cols: [color('色の名前')] → 図形の色
⑥ ★opacity: 数値 → 不透明度（0～1）0が透明、1が不透明
⑦ fade: 'size'/'opacity' → 図形の消え方
'size'は図形が小さくなって消える、'opacity'は図形が透明になって消える（規定値は'size'）

ws_pulse({num: 9, size: [40,70], R: [0.5,1], speed: [5,15]});

ws_pulse({cols: [color('orange')], opacity: [0.5, 1], fade: 'opacity'});

ウィール（ws_whirl）

複数の図形が回転しながら、その輪（円環）を収縮あるいは膨張させます。

個々の図形の大きさ・丸み・色・不透明度は、乱数形式で指定することで、個別に変更することができます。各種速度（speed）や円環の大きさ（diameter）は乱数になりません。
その他、ウィールの場合、以下のようなパラメータがあります。

  • 速度のパラメータ speed に2つの値を指定すると、個々の図形の速度をランダムに変化させるのではなく、円環の [回転速度, 収縮／膨張速度] という意味になります。指定する値が1つだけの場合は「回転速度」を意味します。
  • パラメータ direction に2つの値を指定すると、[円環の収縮／膨張の方向, 左右の回転方向] という意味になります。指定する値が1つだけの場合は「収縮／膨張方向」を意味します。
  • また、収縮と膨張を切り替えて繰り返すか、収縮か膨張のみを繰り返すかをパラメータ fluctuate で指定できます。

① num: 数値 → 図形の数
② ★size: 数値 → 図形の大きさ（1辺あるいは直径の長さ）
③ ★R: 数値 → 角の丸さ（0～1）0が四角、1が円
④ speed: [数値Ａ, 数値Ｂ] → 図形の動く速さ
数値Ａ: 回転速度、数値Ｂ: 収縮/膨張速度
⑤ colors: [color('色の名前')] → 図形の色
⑥ ★opacity: 数値 → 不透明度（0～1）0が透明、1が不透明
⑦ direction: [数値Ａ, 数値Ｂ] → 図形の動く方向
数値Ａ: -1は外から内（収縮）、1は内から外（膨張）に向かって円を描く、数値Ｂ: -1は左回り、1は右回り
数値ＡＢには-1または1を入力
⑧ fluctuate: 'on'/'off' → 収縮と膨張を交互に繰り返すかどうか（規定値は'on'）
⑨ diameter: 数値 → 回転する範囲（直径）width以下の数値

ws_whirl({num: 7, size: [30,80], R: [0.5,1], speed: [5,18], colors: [color('green')]});

ws_whirl({opacity: [0.5, 1],direction: [1, -1],fluctuate: 'off',diameter: [200]});

スパイラル（ws_spiral）

単体の図形が円を描きながら中心あるいは外側に向かって動きます。ウィールの単体図形版です。

図形が単体なので、図形の大きさ・丸み・不透明度の乱数指定は「プログラムを実行するたびに変化する」ことを意味します。ただし、図形の色は円運動が繰り返すたびにランダムに変化します。
その他、ウィールと同様、以下のようなパラメータがあります。

  • 速度のパラメータ speed に2つの値を指定すると、個々の図形の速度をランダムに変化させるのではなく、図形の [回転速度, 収縮／膨張速度] という意味になります。指定する値が1つだけの場合は「回転速度」を意味します。
  • パラメータ direction に2つの値を指定すると、[図形の収縮／膨張の方向, 左右の回転方向] という意味になります。指定する値が1つだけの場合は「収縮／膨張方向」を意味します。
  • パラメータ interval で、図形の色を円運動の途中で変化させる間隔（秒数）を指定できます。このパラメータを省略した場合、円運動の途中で色は変化しません。
  • ウィールとは異なり、パラメータ fluctuate はありません。

① ★size: 数値 → 図形の大きさ（1辺あるいは直径の長さ）
② ★R: 数値 → 角の丸さ（0～1）0が四角、1が円
③ speed: [数値Ａ, 数値Ｂ] → 図形の動く速さ
数値Ａ: 回転速度、数値Ｂ: 収縮/膨張速度
④ colors: [color('色の名前')] → 図形の色
⑤ ★opacity: 数値 → 不透明度（0～1）0が透明、1が不透明
⑥ direction: [数値Ａ, 数値Ｂ] → 図形の動く方向
数値Ａ: -1は外から内（収縮）、1は内から外（膨張）に向かって円を描く、数値Ｂ: -1は左回り、1は右回り
数値ＡＢには-1または1を入力
⑦ diameter: 数値 → 回転する範囲（直径）width以下の数値
⑧ interval: 数値 → 色が変化する間隔
0.5にすると、0.5秒ごとに色が変わる

ws_spiral({size: 70, R: 0, speed: [80, 10], colors: [color('skyblue')] });

ws_spiral({opacity: 0.9, direction: [-1, 1], diameter: 150 });

グリッド（ws_grid）

キャンバスに敷き詰められた図形の色が変化します。また、パラメータの設定によっては大きさが変化してプルプル振動します。

個々の図形の大きさ・丸み・変化速度・色・不透明度は、乱数形式で指定することで、個別に変更することができます。

個数のパラメータ num に指定した値は「図形の数」ではなく「1辺の図形の数」になります（つまり、num×num個の図形が描かれる）。パラメータ interval で色が変わる間隔（秒数）を指定できます。また、パラメータ noise によって図形の大きさをランダムに変化させることができます。

① num: 数値 → 図形の数（1辺あるいは直径の長さ）
画面に9個出したいときは3を入力
② ★size: 数値 → 図形の大きさ（1辺あるいは直径の長さ）
③ ★R: 数値 → 角の丸さ（0～1）0が四角、1が円
④ colors: [color('色の名前')] → 図形の色
⑤ ★opacity: 数値 → 不透明度（0～1）0が透明、1が不透明
⑥ interval: 数値 → 色が変化する間隔
0.5にすると、0.5秒ごとに色が変わる
⑦ ★noise: 数値 → 図形が振動する

ws_grid({size: [1, 40], R: [0.5 ,1], colors: [color('purple')]});

ws_grid({opacity: 0.7, interval: 0.5, noise: 5});

ライン（ws_line）

複数の直線を任意の角度で移動させることができます。

個々の直線の長さ・太さ・移動速度・色・不透明度は、乱数形式で指定することで、個別に変更することができます。ラインでは、図形の数（パラメータ num）は目安です（正確な数にならない）。

直線が移動方向（向かう先）は、パラメータ angle で度数（degree）指定します。12時の方向が 0 で、6時の方向が 180 です。すべての直線は同じ方向に移動します（個別にランダム指定はできない）。

① num: 数値 → 個数
② ★size: 数値 → 直線の長さ
③ ★weight: 数値 → 直線の太さ（規定値は2）
④ ★speed: 数値 → 直線の動く速さ
⑤ cols: [color('色の名前')] → 直線の色
⑥ ★opacity: 数値 → 不透明度（0～1）0が透明、1が不透明
⑦ angle: 数値 → 進行角度（規定値は90）

ws_line({num: 50, size: [1, 100], weight: [5, 30], speed: [20, 60]});

ws_line({cols: [color('green')], opacity: 0.8, angle: 45});
