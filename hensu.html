変数

変数とは簡単にいうと、「数値や文字などの値を入れられる領域」のことです。わかりやすくするため「値を入れられる箱」として、よく例えられます📦

Pythonで、変数を定義・代入するときの記法は以下の通りです。
[ 定義 ]　変数名 = 初期値
[ 代入 ]　変数名 ＝ 代入したい値

それに対して、JavaScriptは…
[ 定義 ]　let　変数名 = 初期値 ;
[ 代入 ]　変数名 ＝ 代入したい値 ;

Pythonの方が書き方がシンプルだな～という印象を受けたと思います💭
JavaScriptでは変数の宣言に「 let 」が必要で、行の末尾には「 ;（セミコロン）」を付ける必要があります。

dia = 60　//定義
y = 80　//定義

circle(100, y, dia)
circle(200, y, dia)

y = 120　//代入（変数ｙの値が120に上書き）
circle(100, y, dia)
circle(200, y, dia)
let dia = 60 ;　//定義
let y = 80 ;　//定義

circle(100, y, dia) ;
circle(200, y, dia) ;

y = 120 ;　//代入（変数ｙの値が120に上書き）
circle(100, y, dia) ;
circle(200, y, dia) ;
