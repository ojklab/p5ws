リスト（宣言と代入）

リスト（配列）は複数の値を変数に詰め込んだもので、同じ型の変数を複数宣言する必要がなく、一つの変数で複数のデータを扱うことができます。
配列の要素は代入によって値を変更することができます。
リストの一番最後に要素を追加したい場合は、Pythonではappendメソッド、Javascriptではpushメソッドを使います。

【 Pythonの場合 】
宣言 ： 配列名 = [値, 値, ... ]
代入 ： 配列名.append(値)
　　 　 配列名.insert(位置,値)

【 Javascriptの場合 】
宣言 ： let 配列名 = [値, 値, ... ] ;
代入 ： 配列名.push(値) ;
　　 　 配列名[インデックス] = 値;

ary = [0, 1, 2]　// 宣言
ary.append ( “end” )　// 代入
print (ary)　// [0, 1, 2,”end”]

ary = [0, 1, 2]　// 宣言
ary.insert (1,”new”)　// 代入
print (ary)　// [0, "new", 2]
let ary = [0, 1, 2] ;　// 宣言
ary.push ( ‘end’ ) ;　// 代入
print (ary) ;　// [0, 1, 2,’end’]

let ary = [0, 1, 2] ;　// 宣言
ary[1] = ‘new’ ;　// 代入
print (ary);　// [0, "new", 2]

リスト（配列の長さ）

Javascriptにおけるリスト（配列）には「自分の要素数」を表す length という特別な変数が用意されています。
width や heightと同じように変数定義しなくても使える変数ですが、width や height と違って配列毎に要素数は異なるので「配列名.length」というように表記します！

（ → Pythonでは、組み込み関数の len関数 を使ってリストの長さ(要素数)を取得できます ）

ary = [1,2,3,4,5]
print len(ary)　// 5
let ary = [1,2,3,4,5];
print(ary.length);　// 5
