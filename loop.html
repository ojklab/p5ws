繰り返し（While文）

繰り返しとは「同じ処理を何度も繰り返すこと」です。
２つある繰り返し構文のうち、まずは「 while文（whileループともいいます）」を紹介します。

【 Pythonの場合 】
while 繰り返し条件 :
　繰り返したい処理（複数行OK）

【 JavaScriptの場合 】
while ( 繰り返し条件 ) {
　繰り返したい処理（複数行OK）
}
となります。

Pythonでは、繰り返し条件の後にコロン「 : 」を付けるのが特徴ですが、JavaScriptでは、繰り返し条件を「 () 」で囲み、「 {} 」の中に繰り返し処理を記述します。

x = 50
cnt = 1　// 10回数えるための変数

while cnt <= 10 :
　circle(x, height /2, 100)
　x += 100
　cnt += 1 　// 変数cntを1ずつ増やす
let x = 50 ;
let cnt = 1 ; 　// 10回数えるための変数

while (cnt <= 10) {
　circle(x, height / 2, 100) ;
　x += 100 ;
　cnt += 1 ; 　// 変数cntを1ずつ増やす
}

繰り返し（for文）

for文は、主に 繰り返し回数が決まっているときに使います。

【 Pythonの場合 】
for cnt in range(1, 11) :
circle(x, height / 2, 50)
x += 100

【 JavaScriptの場合 】
for ( let cnt = 1; cnt <= 10; cnt += 1 ) {
circle ( x, height / 2, 50 ) ;
x += 100 ;
}

↑ カウンタ変数と呼ばれる変数を1つ作成し、それを１つ増やしたり、１つ減らしたりすることによって、条件を満たしている間繰り返すことができます。

for cnt in range ( 1, 11 ) :
circle (x, height / 2, 50)
x += 100
for ( let cnt = 1; cnt <= 10; cnt += 1 ) {
circle ( x, height / 2, 50 ) ;
x += 100 ;
}
